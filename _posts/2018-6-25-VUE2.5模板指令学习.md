最近在学习 VUE，了解下模板指令

--不应该使用箭头函数来使用 method--

### el 挂载点 



```vue
<body>
    <div id="root">
        <!-- <h1>hello {{msg}}</h1> -->
    </div>

    <script>
        let self = this;
        new Vue({
            el: "#root", 
            template:'<h1>hello {{msg}}</h1>',
            data:{
                msg: "world!"
            },
            methods:{
                
            }
        })
    </script>
</body>
```



### v-text

显示内容，v-html 与 v-text 的区别,

```vue
<body>
    <div id="root">
        <h1 v-text="number"></h1>
    </div>

    <script>
        let self = this;
        new Vue({
            el: "#root", 
            data:{
                number: "123"
            },
            methods:{
                
            }
        })
    </script>
</body>
```

v-html

```vue
<body>
    <div id="root">
        <!-- <h1>hello {{msg}}</h1> -->
        <h1 v-html="content"></h1>
    </div>

    <script>
        let self = this;
        new Vue({
            el: "#root",
            data:{
                msg: "world!",
                content:"<h1>hello</h1>"
            },
            methods:{
                
            }
        })
    </script>
</body>
```



### v-on

点击事件,可以缩写为  **@**

```vue
<body>
    <div id="root">
        <button v-on:click="handleClick">点击{{content}}</button>
    </div>

    <script>
        new Vue({
            el: "#root",
            data:{
                content: ''
            },
            methods:{
                handleClick:function(){
                    this.content = "出现";
                }
            }
        })
    </script>
</body>
```

### 属性绑定&双向数据绑定

#### v-bind 

简写   **:**

```vue
<body>
    <div id="root">
        <div v-bind:title="titleInfo">hello world!</div>
    </div>

    <script>
        new Vue({
            el: "#root",
            data:{
                titleInfo: "this is hello world",
                content: ''
            },
            methods:{
            }
        })
    </script>
</body>
```

#### v-model

```vue
<body>
    <div id="root">
        <input v-model:value="content" />
        <div>{{content}}</div>
    </div>

    <script>
        new Vue({
            el: "#root",
            data:{
                titleInfo: "this is hello world",
                content: 'this is content'
            },
            methods:{
            }
        })
    </script>
</body>
```

### watch

侦听器

```vue
<body>
    <div id="root">
        <input v-model:value="content" />
        <div>{{content}}</div>
        <div>{{count}}</div>
    </div>

    <script>
        new Vue({
            el: "#root",
            data:{
                titleInfo: "this is hello world",
                content: 'this is content',
                count: 0
            },
            methods:{
            },
            watch:{
                content: function(){
                    this.count ++;
                }
            }
        })
    </script>
</body>
```

### v-show&v-for

显示，v-if是直接隐藏效率较低，v-show是改变css，效率高。

```vue
<body>
    <div id="root">
        <div v-show="show">hello world</div>
        <button @click="handleClick">toggle</button>
        <ul>
            <li v-for="item of list" :key="item">{{item}}</li>
        </ul>
    </div>

    <script>
        let self = this;
        new Vue({
            el: "#root", 
            data:{
                show: true,
                list: [1, 2, 3]
            },
            methods:{
                handleClick: function(){
                    this.show = !this.show;
                }
            }
        })
    </script>
</body>
```

v-for 循环

```vue
<body>
        <div id="root">
            <ul>
                <li v-for="(item, index) of list" :key="index">{{item}}</li>
            </ul>
        </div>
    
        <script>
            let self = this;
            new Vue({
                el: "#root", 
                data:{
                    list: [1, 2, 3]
                },
                methods:{
                    handleClick: function(){
                        this.show = !this.show;
                    }
                }
            })
        </script>
    </body>
```
